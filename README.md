# Gift List

## Overview

Gift List is an app where you can add people's birthday and gifts. It was made with React, Material UI and date-fns. You can see it [here](https://gift.nahuelclotet.com.ar/).

## Installation

Clone the repository then run:

```bash
npm install
npm run dev
```
